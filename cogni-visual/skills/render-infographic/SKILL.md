# Render Infographic — Dispatcher

Route an infographic-brief.md to the right rendering agent based on its style preset.

Two rendering paths exist — each uses an Opus-powered agent that knows its visual style:

| Style Presets | Agent | Output |
|--------------|-------|--------|
| sketchnote, whiteboard | `render-infographic-excalidraw` | Hand-drawn Excalidraw scene (Mike Rohde / RSA Animate style) |
| economist, editorial, data-viz, corporate | `render-infographic-pencil` | Pixel-precise editorial .pen file (The Economist style) |

## Workflow

### Step 1: Discover Brief

1. If a path argument is provided, use it directly
2. Otherwise glob for `**/infographic-brief.md` (max 3 levels)
3. If multiple found, present candidates via AskUserQuestion
4. Read the brief and validate `type: infographic-brief`, `version: "1.0"`

### Step 2: Determine Renderer

Read `style_preset` from the brief's YAML frontmatter:

- **sketchnote** or **whiteboard** → dispatch to `render-infographic-excalidraw` agent
- **economist**, **editorial**, **data-viz**, or **corporate** → dispatch to `render-infographic-pencil` agent
- If `style_preset` is missing or unrecognized, ask the user which renderer to use

### Step 3: Dispatch Agent

Spawn the appropriate agent with:
- `BRIEF_PATH`: absolute path to the infographic-brief.md
- `OUTPUT_PATH`: user-provided output path, or let the agent default to `{brief_dir}/`

### Step 4: Return Result

The agent returns JSON. Forward it to the user along with:
- Path to the output file (.excalidraw or .pen)
- Share URL (if Excalidraw)
- Element/operation count
